# 描画用のフォーマットについて
Jsonのフォーマットにて記載する

## 問題セット定義ファイル
以下のように記載すること

*questionSet_0001.json*
```
{
    "name": "問題セットの名前",
    "questionSet": [
        {
            "name": "問題1",
            "data": ["問題文1\n正解1"]
        },
        {
            "name": "問題2",
            "data": ["問題文2\n正解2"]
        }
    ]
}
```
